About

One-liner
A free, open-source desktop dashboard for autocannon HTTP load testing — real-time charts, latency percentiles, and full configuration without the CLI.
Short description (100 words)
Autocannon Dashboard is a free Electron desktop app that turns autocannon — the fastest HTTP benchmarking library for Node.js — into a visual, project-driven workflow. Developers create named projects, configure tests through a tabbed UI (no flags, no terminal), hit Run, and watch eight live charts update in real time: requests/sec, throughput, latency percentiles (p50/p90/p99), status codes, errors, cumulative totals, and average response size. Every completed test is stored automatically, browsable as a full report, and exportable to JSON or CSV. It works on Windows, macOS, and Linux. MIT licensed. No account required.
Long description
Autocannon Dashboard is a free, open-source desktop application for HTTP load testing, built around the autocannon Node.js library. It is designed for developers who already know load testing and want a purpose-built workspace instead of a terminal window. The app organises tests into projects. Each project holds multiple named test configurations, which expose the full surface of autocannon's API through a four-tab form: basic settings (URL, method, connections, duration or request count, timeout), headers and body (key-value editor, raw/JSON/form modes), advanced options (pipelining, workers, rate limiting, bailout, reconnect rate, socket path, SNI, expected body), and multi-request scenarios for complex sequential flows. During a test, eight charts update every second from live response data: requests per second, throughput in bytes/sec, four latency percentile lines (p50/p75/p90/p99) on a single chart, req/s vs. mean latency on a dual-axis chart, 2xx vs. non-2xx status codes stacked, errors per second as a bar chart, cumulative request total, and average response size. Eight KPI cards above the charts show the latest tick values at a glance. Every completed run is persisted and accessible through the result history panel. Full reports include a latency percentile histogram, request and throughput statistics, and error breakdowns. Any result can be exported to JSON or CSV. The app also logs structured output to the terminal for CI pipelines: a per-tick table with req/s, throughput, p50, and p99, followed by a summary when the test completes. Autocannon Dashboard runs on Windows, macOS, and Linux, requires no account or network connection, and is licensed under the MIT licence.

Technology

⚡ autocannon 7 🔵 Electron 33 ⚛️ React 18 📦 electron-vite 🎨 Tailwind CSS 3 📊 Recharts 2 💾 electron-store 🔷 Node.js 18+

The main process runs autocannon directly in Node.js and streams per-second snapshots to the React renderer via Electron IPC. Latency percentiles (p50–p99) are computed each second by sorting all response times in the current window — no approximation, no HDR histogram approximation on the client side.
